English:

Hello everyone, I wanted to say thank you to anyone who is using this site and helping to provide feedback. I created this website because I felt that there weren't that many resources to help me learn Romanian. This wasn’t my first language that I chose to learn (I learned Spanish in school). I feel that with languages like Spanish and French you have a lot of resources to help explain, teach, and practice the language. Romanian is definitely not one of those languages that comes with a lot of resources. With this site I hope to provide a place that is filled with knowledge and free so that people can easily learn this beautiful language. This has been a passion project and a chance for me to learn how to create a website so if you notice any issues, please let me know so that I can learn from them and fix them.

For this Update:
  • Extra conjugation practice for the present tense
  • Organization of the files for the website structure has been improved
  • Simplified certain files to make the site more efficient
  • Created this blog

Thank you to everyone who is supporting this site. Please, if you have any suggestions, contact the support email and let us know.

Română:

Bună ziua tuturor, Vreau să zic vă mulțumesc celor care folosiți acest site și care îmi dați feedback. Am creat acest website pentru că am simțit că nu sunt multe resurse pentru învățarea limbii române . Română nu este prima limba pe care am ales să o învăț (am învăț Spaniolă în școală). Simt ca cu limbi că Spaniola si Franceza exista multe resurse care pot să-ți explice, să-i predea, și cu care poți să exersezi limbă. Română nu este una dintre aceste limbi cu multe resurse. Cu acest site, sper să vă dau un mediu plin de informații și să fie gratis, astfel încât oamenii să poată învăța aceasta limba frumoasa. Acest proiect este un proiect din pasiune și o șansă să-nvăț cum se face un website, deci dacă vedeți niște probleme, vă rog sa mă anunțați că să învăț și să le pot repară.
